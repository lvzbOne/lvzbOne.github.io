GitHub+Hugo搭建个人博客

博客搭建

Posted by     "ZhiBiao" on Friday, May 6, 2022

GitHub个人博客搭建

前置条件

  • 采用方案：GitHub + Hugo来实现
  • 不要使用windows 的powershell窗口，使用GitBash来执行命令

常用命令

  • 创建新的内容：hugo new post/XXX.md, 记得修改 draft = false
  • 获取主题：cd themes目录，git clone XXXX ,获取到主题后，把主题的example里的配置文件进行参就行
  • 本地预览：hugo server ,hugo server --theme=hugo-theme-cleanwhite --buildDrafts --watch ，端口可能会变化默认是1313
  • 编译生成文件public下的内容: hugo, hugo --theme=XXXX --baseURL="http://XXXXXX" theme参 数记得使用好自己的主题
  • hugo官网参考命令

（一）新建 GitHub 个人仓库

（二）安装 Hugo

  • 官网下载
  • 查看你的机器架构：echo %PROCESSOR_ARCHITECTURE% 如果是amd的就用amd,如果是arm就下载arm。我的是win11 i5的处理器，看了下是amd的架构,注意下载扩展版的
  • 解压后，配置好环境变量
  • 验证：hugo version
C:\Users\www11>echo %PROCESSOR_ARCHITECTURE%
AMD64

$ hugo version
hugo v0.126.1-3d40aba512931031921463dafc172c0d124437b8 windows/amd64 BuildDate=2024-05-15T10:42:34Z VendorInfo=gohugoio

（三）config.toml文件详解

## 
baseurl = "https://XXXX.github.io/"

## 分页
paginate = 10

[params]
  ## 开启tags的跳转分类
  featured_tags = true
  ## tag的文章>1时出现
  featured_condition_size = 1

## 友情链接配置
[[params.friend_link]]
  title = "XXXXX"
  href =  "https://XXXXXX"  
  
## 不祥
[markup]
  [markup.tableOfContents]
    endLevel = 2
    startLevel = 1
  ## 代码的风格
  [markup.highlight]
    style = "dracula"

（四）md文件头详解

+++
// 浏览页博客的文章标题
title = 'Second'
// 子标题
subtitle = '再次复习Git'
// 发布日期
date = 2023-05-31T18:52:52+08:00
// 是否是草稿，true的话不会展示，false才能浏览展示和发布
draft = false
// 文章作者
author = "XXXX"
// 顶部的图片路径
image = "img/blg-top.jpg"
// 浏览页的细节描述，如果没有设置默认截取文章的部分内容大概好几行的样子
description = "争对实战常用的命令进行滚瓜烂熟，达到进来就能参考，各种场景案例的模拟："
// 标签分类，点击tags时会展示该tags下的所有的文章
tags = ['Git'] 
// 顶部上面的类别
categories = [ 'Tech','Note']
// 打包时用于根据日期好找它对应的目录
URL = "/2023/05/31/Second/"
+++

图床工具

  • 天翼云盘
  • 上传天翼云盘图片后，在网页端打开，然后点击图片，右键复制图片的地址，把该图片放到makrdown中的![图片描述](https://XXXXX)语法里即可

脚本

推送脚本

#!/bin/bash

commit_msg=$1

# 检查参数数量是否为1
if [ "$#" -ne 1 ]; then
    echo "期望传入1个参数，但实际传入了 $# 个参数"
    exit 1
fi

echo " msg: $commit_msg"
# 切换到指定目录
cd /d/Blog/XXXX/

# 执行 hugo 命令
hugo 

# 切换到 public 目录
cd /d/Blog/XXXX/public/

# 执行 git pull 和 git push 命令
git add .
git commit -m "$commit_msg"
git pull
git push origin master:main

本地预览脚本

# 切换到指定目录
cd /d/Blog/XXXX/

# 执行 hugo 命令 hugo-theme-cleanwhite 可以是你的主题名称
hugo server
